# HTTP Protocol Implementation

Complete HTTP protocol implementation including client, server, message handling, and parsing.

## Navigation

- [Parent Directory](../README.md)
- [Repository Root](../../README.rst) (or navigate up)

## Subdirectories

- **algorithms/**: HTTP algorithms
- **client/**: HTTP client implementation
- **impl/**: HTTP implementation details
- **message/**: HTTP message types
- **parser/**: HTTP parser implementation
- **policies/**: HTTP policy implementations
- **server/**: HTTP server implementation
- **support/**: HTTP support utilities
- **traits/**: HTTP type traits
