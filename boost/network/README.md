# Network Library Core

Core network library implementation including message types, protocol implementations, URI handling, and utility functions.

## Navigation

- [Parent Directory](../README.md)
- [Repository Root](../../README.rst) (or navigate up)

## Key Files

- **constants.hpp**: Network constants and definitions
- **message.hpp**: Generic message type implementation
- **protocol.hpp**: Protocol definitions
- **uri.hpp**: URI handling interface
- **version.hpp**: Library version information

## Subdirectories

- **detail/**: Internal implementation details
- **message/**: Message type implementations
- **protocol/**: Network protocol implementations
- **uri/**: URI parsing and handling
- **utils/**: Utility functions and helpers
