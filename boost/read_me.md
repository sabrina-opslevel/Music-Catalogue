# Boost Directory

## Description

This directory contains the Boost-style header files for the cpp-netlib project. These headers provide the main entry points for including the network library functionality in your C++ projects.

## Contents

- `network.hpp` - Main header file that includes all network-related functionality
- `mime.hpp` - MIME type handling utilities
- `network/` - Directory containing the detailed network protocol implementations

## Usage

To use the cpp-netlib in your project, include the appropriate header file:

```cpp
#include <boost/network.hpp>
```

For more information about building and using cpp-netlib, please refer to the main project README.
