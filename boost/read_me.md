# Boost Directory

## Description

This directory contains the Boost-style header files for the C++ Network Library. It provides the main interface headers for network and MIME functionality.

## Contents

- `network.hpp` - Main header for network functionality
- `mime.hpp` - MIME type handling utilities
- `network/` - Directory containing detailed network implementation headers

## Usage

Include the appropriate header files in your C++ projects:

```cpp
#include <boost/network.hpp>
#include <boost/mime.hpp>
```

These headers provide the core networking abstractions and utilities from the cpp-netlib library.
